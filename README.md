# Code for Mamba-Trident: Terrain-guided Tri-modal Mamba Network for Mapping Little Ice Age Glaciers

Code accompanying the manuscript currently under review.

## Contents

- `src/models/` — proposed model architectures
- `src/ablation_engine/` — ablation framework
- `src/utils/` — loss functions and evaluation metrics

## Dependencies

- PyTorch 2.x
- [mamba-ssm](https://github.com/state-spaces/mamba) >= 2.3.1
- [VMamba](https://github.com/MzeroMiko/VMamba) (used as backbone)

## Note

This repository contains the core implementation referenced in the paper.
Training scripts and data loading utilities are omitted.
Author and institutional information are withheld during peer review.
